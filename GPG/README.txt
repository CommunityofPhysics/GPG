GitHub repository for the Gravitational Physic Group, Division of Experimentation, Community of Physics.

*** Code of conduct ***
  1.  This folder and its subfolders contain the codes and computational equipments created by the Gravitational Physics Group of Community of Physics. 
  2.  This folder contains materials that belong only to the Gravitational Physics Group. 
  3.  Files that belong to this group must not be uploaded/submitted in another groups's folder.
  4.  Files must not be created, edited, modified uploaded or deleted without permission from the proper authority. 
  5.  Failure to abide by these rules may result in stringent action. 
  6.  All approved codes from projects are to be uploaded in the Github repository as a standard practice of Community of Physics.
